# Tech Stack
The website is developed using MERN stack.
* MongoDB: A document-oriented, NoSQL database
* Express.js: A backend web application framework for Node.js
* React.js: A frontend library commonly used for single-page applications.
* Node.js: A server-side JavaScript runtime.
